# coding-1-escape-room-project-part-1
Part I: In this project, you will design and implement a single text-based puzzle using Python functions (think escape room puzzle). Your focus should be on creating clear, modular code that correctly handles input, produces the expected outputs, and demonstrates thoughtful function design.

Project Requirements:

## 1. Functions & Modular Design (15 pts)
 - Program uses at least 3 functions, each with a clear purpose.
 - Functions use parameters and return values appropriately.
 - Code is organized, readable, and free of duplicate logic.

## 2. Puzzle Logic & Input Handling (10 pts)
 - Puzzle correctly processes user input and produces expected outputs.
 - Handles at least one type of invalid input gracefully.
 - Includes at least one conditional (if/else) and optional loop.
 - Uses at least one dictionary to store puzzle state, clues, or player progress.

## 3. Player-Facing Interface - Console (15 pts)
 - Provide a welcome message or instructions so a new player knows how to play your puzzle
 - Display the current puzzle or room clearly to the player
 - Show success/failure or progress after the puzzle is attempted

## 4. Optional Bonus (+10 pts Each)
 - Adds a creative twist, hidden feature, or randomized element
 - Design your puzzle so there is more than one correct way to solve it.
 - Add a feature that shows player time / gives a score for better performance.
